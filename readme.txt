Add a Profile Image field to BuddyForms !

== Description ==

### Collect user Profile Image during Registration
> * Create outstanding Registration forms and ask your users to upload a profile image.


### Use the profile image form element in any form
> * A new form element "Profile Image" is available in the form element select and can be used with any form type.


### BuddyPress and Ultimate Member Compatible
> * Works with WordPress standard alone or in combination with BuddyPress or Ultimate Member.
